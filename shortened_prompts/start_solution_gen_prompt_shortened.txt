## ГЕНЕРАЦИЯ РЕШЕНИЯ

Разработай структурированное решение задачи, используя подход, соответствующий её типу. Для математических задач применяй пошаговые вычисления, для программирования - обоснованный алгоритм, для творческих - рассмотри альтернативы. Поясняй сложные моменты, указывай ограничения и допущения. Структурируй ответ, выделяя ключевые этапы и результаты. Обеспечь соответствие ранее определенным критериям качества.
